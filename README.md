# CalculationHours
Утилита для подсчета количества часов в соответствии с требовании организации.
Одностраничный сайт, где пользователь подгружает эксель-файл, вводит параметры для расчетов, так как файл содержит огромное количество данных.
При нажатии на кнопку "Рассчитать" - утилита выдает эксель файл со списком сотрудников и подсчитанным количеством часов за выбранный диапазон
